# Quintin 2014 — 7 Things You Should Know About Tor

+ Tor still works:
  * NSA leaks showed that Tor has not been completely broken.
  * Most successful attacks come from browser vulnerabilities, user mistakes, or traffic analysis rather than breaking Tor’s encryption.
+ Tor is not only used by criminals:
  * Many people use Tor for legitimate reasons.
  * Examples include journalists, activists, researchers, families, and the military.
  * Tor helps protect privacy
+ Tor does not contain a military backdoor:
  * Although early development was funded by the U.S. Navy, now Tor is open-source.
  * Security experts and cryptographers have examined the code and found no backdoor.
+ Running a Tor relay is not illegal in the U.S.:
  * According to the EFF, no one in the U.S. has been prosecuted simply for running a Tor relay.
  * However, exit relays may attract attention from law enforcement.
+ Tor is easy to use:
  * Users can simply download the Tor Browser Bundle, which is pre-configured for secure use.
  * Another option is Tails, a privacy-focused operating system that routes all traffic through Tor.
+ Tor is slower than normal internet but improving:
  * Tor routes traffic through multiple relays, which causes some slowdown.
  * However, the network has been improving and becomes faster.
+ Tor is not foolproof:
  * Users can lose anonymity if they use Tor incorrectly.
  * Logging into personal accounts like Google or Facebook can reveal identity.
  * Advanced attackers may still perform traffic analysis.

# Shavers & Bair 2016 – Hiding Behind the Keyboard: The Tor Browser

## Introduction
+ Tor is a modified version of the Firefox browser designed to hide the user’s real IP address.
+ Its main purpose is to allow people to communicate and browse the internet anonymously.
+ Tor is easy to use, even for people without advanced technical skills.
+ Because it is simple, free, and effective, Tor can be used for both legitimate and illegal purposes.
